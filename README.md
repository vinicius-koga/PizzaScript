# 🍕 PizzaScript

Um projeto desenvolvido com **HTML, CSS e JavaScript** que simula o funcionamento de uma pizzaria online. O objetivo principal foi praticar manipulação do DOM, organização de código, módulos JavaScript e consumo de dados em formato JSON.

---

## 📖 Sobre o projeto

O **PizzaScript** é uma aplicação web onde o usuário pode navegar por um catálogo de pizzas, visualizar seus detalhes, adicioná-las ao carrinho e alterar a quantidade de itens antes de finalizar o pedido.

O principal foco do projeto foi tornar o catálogo totalmente dinâmico. Todas as informações das pizzas são carregadas a partir de um único arquivo que simula uma resposta JSON. Dessa forma, é possível adicionar, remover ou editar pizzas sem modificar o restante do código da aplicação.

---

## ✨ Funcionalidades

- 📋 Exibição dinâmica do catálogo de pizzas
- 🔍 Visualização de detalhes da pizza
- 🛒 Adição de pizzas ao carrinho
- ➕ Aumento da quantidade de itens
- ➖ Diminuição da quantidade de itens
- ❌ Remoção automática de itens quando a quantidade chega a zero
- 💰 Cálculo automático do subtotal
- 🚚 Cálculo de desconto
- 💵 Cálculo do valor total do pedido
- 📱 Interface responsiva
